FLUJO DE USUARIO:

1. USUARIO VE CRUCIGRAMA
   ├─ Cuadrícula 7×7 a la izquierda
   ├─ Pistas organizadas (Horizontales/Verticales) a la derecha
   └─ Botones "Verificar Crucigrama" y "Limpiar"

2. USUARIO COMIENZA A ESCRIBIR
   ├─ Hace click en una celda → se selecciona (naranja)
   ├─ Escribe una letra → aparece en la celda
   ├─ Auto-avanza a siguiente celda
   └─ Puede usar flechas para movimiento manual

3. USUARIO COMPLETA PALABRAS
   ├─ Navega con flechas y escritura
   ├─ Si hace click en pista → resalta esa palabra
   ├─ Continúa hasta llenar todo
   └─ Los números guían qué palabra es cada posición

4. USUARIO VERIFICA
   ├─ Click en "Verificar Crucigrama"
   ├─ Sistema compara con grid correcto
   ├─ ✅ SI CORRECTO → Video desbloqueado
   └─ ❌ SI INCORRECTO → Mensaje de error

5. SI CORRECTO
   ├─ Mensaje: "¡CORRECTO! 🎉 Desbloqueando..."
   ├─ Crucigrama desaparece
   ├─ VIDEO desbloqueado con animación
   ├─ Badge: "🎁 ¡DESBLOQUEADO!"
   └─ Botones para compartir en redes
